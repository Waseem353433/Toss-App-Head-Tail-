# Toss-App-Head-Tail-
🪙 Coin Toss Application
📌 Project Description

This is a simple Coin Toss Application built using HTML, CSS, and JavaScript.
The app simulates a real coin flip. When the user clicks a button, it randomly displays either "Head" or "Tail" on the screen.

🚀 Features
🎲 Random coin toss result (Head or Tail)
🖱️ Button click to toss the coin
⚡ Instant result display using DOM manipulation
🎨 Simple and clean user interface
🛠️ Technologies Used
HTML – Structure of the application
CSS – Styling and layout
JavaScript – Logic and random result generation
